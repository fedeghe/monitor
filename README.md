# web-page-monitor (work in progress)

### install
``` sh
yarn add web-page-monitor
```


### use it
```
import 'web-page-monitor'

WebPageMonitor
    .showFPS()
    .showMEM()
    .showTAGS()
    .showEVENTS()
    .showNET()
    .render();
```
